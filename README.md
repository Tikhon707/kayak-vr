# rowing-vr
https://github.com/tehnik58/rowing-vr.git

    Реалистичный симулятор гребли на каяке в VR с физикой плавучести, динамическим центром масс и двойным веслом. Поддержка OpenXR / XR Interaction Toolkit. Unity 2022+.

## Особенности
* Физика плавучести: каяк реалистично погружается, кренится и качается в зависимости от наклона игрока.

* Динамический центр масс: смещается при наклоне головы в VR — вызывает естественный крен и дифферент.
* Двойное весло: гребля двумя руками с учётом глубины, скорости и угла лопасти.
* Полная поддержка VR: работает с любыми контроллерами через XR Interaction Toolkit.
* Модульная архитектура: легко настроить под любую модель каяка или весла.

## Требования
* Unity: 2022.3 LTS или новее
* Пакеты:
    * com.unity.xr.interaction.toolkit (2.5+)
    * com.unity.inputsystem (1.7+)
    * com.unity.openxr (или другой XR-плагин по выбору)
* Платформа: PC VR (Quest через Link/Air Link, Valve Index, HTC Vive и др.)

## Установка
1. Клонируйте репозиторий:

    >git clone https://github.com/tehnik58/rowing-vr.git

2. Откройте проект в Unity.
2. Убедитесь, что установлены все зависимости (через Window > Package Manager).
2. Откройте сцену: ```Assets/Scenes/SampleScene.unity```.

## Настройка под свою модель
Проект легко адаптируется под любую модель каяка или весла.
### Каяк
* Замените модель в объекте Kayak.

* Настройте BuoyancySystem:
    * Обновите points под габариты модели.
    * Подберите area, чтобы суммарная плавучесть ≈ весу (mass * 9.81).
* Убедитесь, что Rigidbody.mass = 80–100.

### Весло
``` DoublePaddle
├── LeftBlade
│   └── BladeTip
└── RightBlade
    └── BladeTip
```
* ```BladeTip``` — в кончике лопасти.
* ```LeftBlade.up``` и ```RightBlade.up``` должны быть перпендикулярны плоскости лопасти.
